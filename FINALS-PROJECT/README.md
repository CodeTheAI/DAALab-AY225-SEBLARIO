# Wine Reviews Data Visualization

A collaborative data science project for visualizing and analyzing wine review data.

## Project Overview

This project analyzes the Wine Reviews dataset, which contains wine reviews from various regions around the world. The visualization includes interactive charts, data tables, and summary statistics to help understand wine quality, pricing, and regional distributions.

## Dataset

- **Source**: Wine Magazine dataset
- **Records**: ~130,000 wine reviews
- **Fields**: Wine name, country, region, price, points (rating), variety, winery, description

## Features

- Interactive data table with sorting and filtering
- Summary statistics dashboard
- Price vs. Quality scatter plot
- Regional distribution charts
- Wine variety analysis
